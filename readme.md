Một số đề cơ bản để làm quen

Bài 1 — Tính tổng 2 số

Viết chương trình nhập 2 số nguyên a, b rồi in ra tổng của chúng.

Input
3 5
Output
8



Bài 2 — Tính diện tích hình chữ nhật

Nhập chiều dài và chiều rộng, in ra diện tích.

Input
4 7
Output
28

Bài 3 — Tìm số lớn nhất trong mảng

Nhập:

số lượng phần tử n
mảng gồm n số nguyên

In ra số lớn nhất.

Input
5
3 9 2 7 1
Output
9



Bài 4 — Đếm số chẵn

Nhập mảng n số nguyên, đếm xem có bao nhiêu số chẵn.

Input
6
1 2 4 7 8 9
Output
3


Bài 5 — Sắp xếp tăng dần

Nhập mảng n số nguyên và sắp xếp tăng dần bằng Bubble Sort.

Input
5
5 1 4 2 3
Output
1 2 3 4 5
